Vending Machine
Приложение выполнено на .NET Core 3.0

/Client/Index - пользовательский интерфейс.
/Admin/Panel/SecretKey или /Admin/Panel?key=SecretKey - административный интерфейс.
"SecretKey" - секретный ключ.

Файл с базой данных VendingMachine.mdf, располагается в папке /VendingMachine/App_Data.

Необязательные требования:
Возможность покупки нескольких напитков перед получением сдачи.
